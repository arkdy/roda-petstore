# roda-petstore

Sample petstore to test rswag-rack gem

## Installation

1. clone repo
2. `bundle install`

### Run specs

- `bundle exec rspec`

### Generate OpenApi json

- `bundle exec rake`

see docs in newly created `swagger` directory

### Run Roda app

- `bundle exec rackup app.rb`

## Where to see details

- see `spec/spec_helper.rb`
- `spec-api_spec.rb`
